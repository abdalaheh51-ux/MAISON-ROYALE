تثبيت حزم المشروع
bun install
---------------
إعداد قاعدة البيانات
# إنشاء قاعدة البيانات + الجداول
bun run db:push

# زرع بيانات الساعات + المخزون (18 مكوّن)
bun run scripts/seed.ts
-------------------------------
ملف .env جاهز بالفعل بمسار نسبي:
DATABASE_URL=file:./db/custom.db
------------------------------------
تشغيل خادم التطوير
bun run dev
----------------------------
ثم افتح المتصفح على:
http://localhost:3000
